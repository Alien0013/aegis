# AEGIS — ਪੰਜਾਬੀ ਝਲਕ

ਇਹ ਪੰਨਾ ਇੱਕ ਸਥਾਨਕ ਭਾਸ਼ਾ ਸਨੈਪਸ਼ਾਟ ਹੈ। ਸਹੀ ਕਮਾਂਡਾਂ, ਬਣੇ ਹੋਏ ਰੈਫਰੈਂਸ, ਅਤੇ ਤੇਜ਼ੀ ਨਾਲ ਬਦਲਣ ਵਾਲੇ ਵੇਰਵਿਆਂ ਲਈ ਅੰਗਰੇਜ਼ੀ ਦਸਤਾਵੇਜ਼ ਕੈਨੋਨਿਕਲ ਹਨ।

## ਮੁੱਖ ਗੱਲਾਂ

- AEGIS ਇੱਕ local-first agent harness ਹੈ ਜਿਸ ਵਿੱਚ CLI, dashboard, desktop app, messaging gateway, API, ACP, MCP, scheduled work, memory, ਅਤੇ skills ਸ਼ਾਮਲ ਹਨ।

- ਹਰ contribution ਲਈ tests, generated reference updates, ਅਤੇ sensitive output redaction ਲੋੜੀਂਦੇ ਹਨ।
