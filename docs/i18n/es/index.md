# AEGIS — resumen en español

Esta página es una instantánea localizada. La documentación en inglés sigue siendo la fuente canónica para comandos exactos, referencias generadas y detalles que cambian con frecuencia.

## Puntos clave

- AEGIS es un arnés de agente local-first con CLI, panel web, app de escritorio, pasarelas de mensajería, API, ACP, MCP, trabajos programados, memoria y habilidades.
- Los secretos no se publican en las guías. Las variables de entorno se documentan sin valores reales.
- La prueba local y la prueba live externa son diferentes: un adaptador falso no demuestra entrega real por Telegram, Discord, SMS, email o un proveedor de modelos.
- Cada cambio debe incluir pruebas, referencias regeneradas cuando aplique, y redacción de datos sensibles.
