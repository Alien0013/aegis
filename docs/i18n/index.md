# AEGIS i18n status

AEGIS documentation is authored in English first and exposes localized snapshot entry points so translation work is visible, reviewable, and testable.

## Locales

| Locale | Status | Scope |
| --- | --- | --- |
| English (`en`) | Canonical | Full docs, generated references, release notes, and maturity reports. |
| French (`fr`) | Snapshot | Install, quickstart, runtime surfaces, safety model, and contribution path. |
| Spanish (`es`) | Snapshot | Install, quickstart, runtime surfaces, safety model, and contribution path. |
| Simplified Chinese (`zh-Hans`) | Snapshot | Install, quickstart, runtime surfaces, safety model, and contribution path. |
| Punjabi (`pa`) | Snapshot | Install, quickstart, runtime surfaces, safety model, and contribution path. |

## Rules

- English remains the canonical source until a locale is marked complete.
- Localized pages must link back to canonical docs when details can drift.
- Security, credentials, and live-QA language must not be softened in translation.
- Generated references are not manually translated; translate the narrative guides and link the generated reference.
