# AEGIS — aperçu français

Cette page est un instantané localisé. Les pages anglaises restent la référence canonique pour les commandes exactes, les options générées, et les détails qui changent souvent.

## Points clés

- AEGIS est un harnais d'agent local-first avec CLI, tableau de bord, application desktop, passerelles de messagerie, API, ACP, MCP, tâches planifiées, mémoire et compétences.
- Les secrets restent hors des guides publics. Les variables d'environnement sont décrites sans valeur réelle.
- Les preuves locales et les tests live externes sont séparés: un adaptateur simulé ne prouve pas une livraison Telegram, Discord, SMS, email ou provider réelle.
- Les nouvelles contributions doivent ajouter des tests, régénérer les références, et garder les sorties sensibles masquées.
